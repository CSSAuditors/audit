CSS File Sizes Audit Report

#Respect CSS

CSS Auditors have one mission: to put the CSS under the spotlight and educate developers about its importance.

From file size to usage of particular attributes and values, we try to understand ways how to improve the way CSS could and should be written.

Sizes Report

In this report we are going to focus on CSS filesize. The aim of this report is to understand how much CSS code is needed to build a site.

What We Learned

Like in any other language, the file size matters. This report shows that the size of the CSS is not insignificant. In fact, 5 sites have more than 1MB of External CSS code, and 1 site have more than 2MB of External CSS code.

Style tag CSS is

We noticed that most sites use Inline CSS. While Inline CSS is not considered the best practice (you get the warning for using inline CSS in Google Lighthouse tool, for example), it is still heavily used by developers of these sites.

Different sites use different strategies to deliver CSS. No matter which way it is used, we should aim to deliver the least possible amount of code to the end user.
